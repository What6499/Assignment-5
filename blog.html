Question-1: What are the different ways to select an element in the DOM?

In DOM we can select an element using its id, class or tagName using js methods, getElementById(), getElementsByClassName() and getElementsByTagName(). We can also get elements just like we do in CSS using CSS Selectors, querySelector() and querySelectorAll() in js.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML will give you everything inside an element. That includes all the HTML tags and comments as well.

2. innerText only gives you the visible texts inside the element. There will be no Comments nor HTML Tags.

3. textContent pretty much the same as innerText. The only addition is that it shows comments as well.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in which, instead of adding event listeners to each element, you add a single event listener to a parent element to handle events on its child elements.

Question-4: What is event bubbling in the DOM?

If you click on a button, the event that is triggered on that element will propagate through its parent elements until it reaches the root of the document. Essentially it 'bubbles-up' until it reaches the root. This allows parent elements to handle events from their child element.

Question-5: How do you create, add, and remove elements using JavaScript?

In js we can create elements using document.createElement('TagName') and put it in a variable. Now, to add them in the DOM we can use .appendChild() to place them within a parent element. And now, if we wanna remove that element we can use .remove() on on the variable we put the created element on or we could call .removeChild() on a parent element to remove it.
